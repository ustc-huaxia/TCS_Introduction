# USTC TCS

## 简介
本仓库针对中国科学技术大学华夏计算机英才班科学研讨课建立，旨在为学生们提供理论计算机方向(TCS)的相关资料与经验，并整理留存每一届学生在科学与社会研讨课中的TCS方向的成果。
